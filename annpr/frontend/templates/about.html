ABOUT ANNPR

ANNPR(Automatic Nepali Number Plate Recognition System) is a system designed to detect Nepali number plates from images or videos and then to recognize the number plate. In this system we used YOLOv4 Darknet model for detecting number plates as well as for character segmentation. Then the segmented characters are recognized using a CNN model. The recognized license plate number will then be recorded in a database.
